Layout Css

display é a propriedade css mais importante para controlar o layout.
layout, ou leiaute, é o modo de distribuição de elementos em um determinado espaço.
Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento. A maioria dos elementos é em bloco (block) ou inline.

display: inline pode ser usado para modificar o estilo bloco por inline, e o contrário também é verdade.
Isso pode fazer uma lista ficar um item do lado do outro ou até um span virar bloco.

display: none
pode ser usado para ocultar um elemento. Ele não irá aparecer e nem ocupará espaço algum.

visibility: hidden
diferente do display: none, ele não irá aparecer, mas ocupará o espaço que ele ocuparia normalmente.
