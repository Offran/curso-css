Fontes

Fontes são usadas para dar certo visual de identidade para seu site, um padrão importante de ser estabelecido.
Usando font-family para especificar a fonte de um texto.
A font-family também deve conter vários substitutos, para garantir máxima compatibilidade entre navegadores/sistemas operacionais. Começando com a desejada, e respectivamente as substitutas.

Por exemplo: p#font-family1{font-family: "Arial, Times New Roman", Times, serif;}

As fontes tem 5 famílias principais:

  • serif
  • sem serifa
  • minivan
  • cursiva
  • fantasia
Para mais detalhes e diferentes fontes confira aqui.

Também é possível utilizar fontes através de links. Usando o Google Fonts:
Adicionando um link:
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Audiowide">
p#font-family2 { font-family: "Audiowide", sans-serif; }

font-style, font-weight e font-variant

Font-style usada para especificar texto em itálico, oblícuo (semelhante ao itálico, mas menos suportado).
Font-weight é a espessura da fonte do texto.
Font-variant é usada para deixar o texto mais achatado, como se fosse minúsculo, mas não dando essa semântica de letra minúscula.
Nesse "p" foi usado:

p#fonte-personalizada1 {
font-style: italic;
font-weight: 900;
font-variant: small-caps;
}

Mas também poderia ter sido usado weight "bold" ou style "oblique".

font-size

É para gerenciar o tamanho da fonte, por exemplo:
p#font-size1{font-size: 2em;}

Nota: o tamanho padrão das fontes de texto normal é 16px = 1em.
O tamanho da fonte não atribui semântica para o navegador, então de preferência não deve ser usado para títulos e subtítulos.

Abreviação de fonte:
p.abreviacao { font: italic small-caps bold 12px/30px Georgia, serif; }
