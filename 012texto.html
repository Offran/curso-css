text-align

O alinhamento de texto pode ser usado de várias formas para ajusta-lo.
exemplos:
text-align: right;
text-align: center;
text-align: left;

text-align: justify
Todas as frases ditas aqui estarão limitadas ao mesmo tamanho.
Então elas, mesmo com palavras diferentes, irão se apertar ou alargar para ficarem num texto quadrado.
Como pode ser observado, o uso do width é importante para limitar a largura e o justify ter o efeito esperado. Usos de <br> também ajudam o texto a ficar num pequeno bloco e mais fácil de ler.

Nota: todas as frases estarão limitadas pela largura do elemento, então o elemento span ou um elemento apertado por um width mais contido não irão até o limite da página, como no exemplo acima.

text-align-last: right; Usada para que a última linha fique na direita. A especificação do valor e o que dirá onde a última linha vai estar.


vertical-align

É para definir um alinhamento vertical de imagens.

Uma imagem na linha (baseline).
Uma imagem com super alinhamento (super).
Uma imagem na parte inferior (text-bottom).
Além disso ainda podemos usar outros valores, como sub e text-top.

decoração de texto

A text-decoraction-line é uma propriedade usada para adicionar uma linha de decoração ao texto.
Também é possível adicionar uma cor (text-decoraction-color), espessura (text-decoration-thickness), estilo (como os explicados em bordas.), a forma abreviada e até removendo qualquer decoração que já venha por padrão (text-decoration: none).

{text-decoration: overline;
text-decoration-color: red;}
{text-decoration: line-through;
text-decoration-thickness: 5px}
{text-decoration: underline; text-decoration-style: dotted;}
{text-decoration: overline underline red 3px;}
{text-decoration: none;} Geralmente no html os links vem sublinhados, mas com isso, eles perdem essa obrigação, como aqui.


Transformação de texto

A text-transform é uma propriedade usada para especificar letras maiúsculas e minúsculas no texto.
Podemos transformar tudo em maiúscula, minúscula ou só as primeiras letras.

{ text-transform: uppercase; }
{ text-transform: lowercase; }
{ text-transform: capitalize; }


Espaçamento de texto

O text spacing possui algumas propriedades que serão listadas:

  • text-indent
    • text-indent é usado para especificar um recuo da primeira linha de texto. Assim podeendo criar um parágrafo. No exemplo daqui está sendo usado {text-indent: 20px;} e então o
  • letter-spacing
    • {letter-spacing: 3px;} é para especificar espaçamento entre cada letra.
  • line-height
    • {line-height: 2px;} é usada para especificar espaço entre as linhas.
    As linhas ficam praticamente juntas quando é colocado 1px.
  • word-spacing
    • {word-spacing: 5px;} é usado para especificar espaçamento entre cada palavra.
  • white-space
    • {white-space: nowrap;} deixa o texto em branco. Aqui eu não farei isso, pois o intuito é essa informação ser lida.


Sombra de texto

A text-shadow adiciona uma sombra ao texto, mas a sombra pode ser usada de várias formas.

{text-shadow: 2px 2px;}

{text-shadow: 2px 2px red;}

{text-shadow: 2px 2px 5px red;}

{text-shadow: 0 0 3px red}

{text-shadow: 0 0 3px blue , 0 0 5px red}

{text-shadow: 1px 1px 2px black , 0 0 25px blue, 0 0 5px darkblue;
color: white; }
