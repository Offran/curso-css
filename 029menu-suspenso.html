Menu suspenso

Menu suspenso pode deixar texto, imagem ou mesmo uma barra de navegação suspensa. É uma forma de compactar e modernizar a estilização do site.

Passe o mouse por mim
.dropdown {
position: relative;
display: inline-block;
}

Respectivamente: ajuste de posição relativa e ajuste para ser um bloco sem quebra de linha.

.dropdown-content {
display: none;
position: absolute;
background-color: #f9f9f9;
min-width: 350px;
box-shadow: 0px 8px 16px 0px rgba(0, 0, 0, 0.2);
padding: 12px 16px;
z-index: 1;
}

Respectivamente: o display remove a aparição do elemento, a posição está relativa ao seu antecessor(pai), background deu cor, min-width dá espaço mínimo, box-shadow estilizou com sombra, padding ajustou espaço dentro do elemento e z-index: 1 é para sobrepôr o elemento.

.dropdown:hover .dropdown-content { display: block; }

Ajuste para fazer um bloco quando passar o mouse por cima, ajustando o anterior "display:none"
