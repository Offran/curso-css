Links Css

Os links em css podem ser estilizado com qualquer uma das propriedades css (color, font-family, background, etc.)
exemplo 1, com bordas exemplo 2, com background color

Além disso, um link pode ter estilos diferentes dependendo do estado em que se encontram:

    a:link
    um link normal não visitado, quando o mouse passa por cima .
    a: visited
    Um link que o usuário visitou
    a:hover
    Um link quando o usuário passa o mouse sobre ele
    a:active
    Um link no momento em que é clicado

Exemplo prático:
Este é o link
Note que ele é verde por padrão: a:visited{color: green}
Quando passo o mouse por ele, ele fica vermelho: a:hover{color: red}
Quando estou pressionando o mouse para clicar nele, sua cor fica azul: a:active{color:blue}

Nota: :hover deve vir depois de :visited; :active deve vir depois de :hover. Tudo que for aplicado assim irá se manter para o site inteiro.

Botões de link

Aqui tem um exemplo bem mais avançado, onde é feito um botão de link com várias propriedades css já vistas até então.

a:link, a:visited {
background-color: #f44336;
color: white;
padding: 14px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}

a:hover, a:active {
background-color: red;
}
