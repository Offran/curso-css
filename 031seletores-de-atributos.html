Seletores de Atributos Css

São muitos os atributos css que servem para estilizar elementos html, será abordado boa parte.

a[target]{background-color: yellow} link amarelo
target="valor"
target~="valor" (todos os títulos serão influenciados)
[class|="top"] {background: yellow;} (todas as classes com hífen serão influenciadas)

Isso funciona para Seletor CSS [atributo$="valor"];
Seletor CSS [atributo^="valor"];
Seletor CSS [atributo*="valor"];
E assim sucessivamente para todos os tipos de seletores css, como aqui.
