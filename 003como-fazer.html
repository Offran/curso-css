Como fazer CSS

Css externo:
Podemos fazer aplicando um estilo externo. É a melhor forma de trabalhar com um CSS muito grande.
<link rel="stylesheet" href="style.css"> (então você deve criar um arquivo com esse nome e iniciar sua estilização).

Css interno:
Ao invés de criar um link, basta abrir uma tag <style<</style> no head.

Css embutido:
<h1 style="propriedade: valor;propriedade:valor;">Título 1 </h1>
<h1 style="color: red;text-align:center;">Título 1 </h1>
Essa é uma demonstração de como funciona o css embutido, com a sintaxe explicativa e prática.

Nota: Caso sejam feitos ambos os casos, Css externo e interno, aquele que tiver sido feito antes terá prioridade.
Fora isso, a ordem é em cascata, nesse estilo:

  1. Estilo Embutido
    2. Em linha (in-line).
  2. Interno ou externo
    3. O que vier primeiro.
  3. Padrão do navegador
    4. Apenas caso não tenha css sobre ele, então o css padrão do seu navegador irá ficar em vigência.

Nota: Para colocar comentários no css é essa a sintaxe:
/* comentários aqui dentro. */
Esses comentários podem ter mais de uma linha.
