Combinadores Css

Um seletor css pode conter vários tipos de seletores apra especificar/dar alvo melhor em cada elemento.
Existem quatro combinadores

  • Seletor descendente (espaço)
    • div p {color: blue} Para aplicar em todos os p descendente de divs.
  • Seletor filho (>)
    • div > p {color: blue} Ele é aplicado assim como o descendente, mas restrito ao filho.
  • Seletor de irmão adjacentes (+)
    • div + p {color: blue} Os elementos irmãos adjacentes devem ter o mesmo pai e vir imediatamente um após o outro. Isso significa que se tiver como no exemplo citado, um p depois do p, ainda que irmão, não será referenciado.
  • Seletor geral de irmãos (~)
    • div ~ p {color: blue} Seleciona todos os irmãos que vierem após o elemento referenciado.
