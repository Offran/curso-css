Esboço/outline

outline é como se fosse uma borda além da borda normal. Ele está fora do elemento e da borda. Além disso, o outline pode se sobrepor a outros conteudos.

p { border: 2px solid black; outline: #4CAF50 solid 10px; margin: auto; padding: 20px; text-align: center; }

Ainda temos algumas propriedades para uso com outline:

  • outline-style
    • outline-style esepcifica o estilo de contorno e pode ter os mesmos valores das bordas.
  • outline-color
    • Usado para definir uma cor. outline-color: red;
  • outline-width
    • Usado para definir uma largura. outline-width: 12px;
  • outline-offset
    • Usado para definir um espaço (padding) entre a borda e a outline. outline-offset: 5px;
  • outline
    • Usando apenas "outline" podemos definir a width, style, color, valroes das bordas, etc. É uma forma abreviada.
