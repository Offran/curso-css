Galeria de imagens Css

Basicamente aqui é para ajustar imagens com descrições e links. Atribuindo melhores características a elas e modernizando o site.

A forma de fazer é a seguinte:
div.gallery { margin: 5px; border: 1px solid #ccc; width: 180px; float: left; }

Respectivamente: margem, borda, largura e flutuação para a esquerda da imagem, assim sendo, a sequência de imagens começa pela esquerda e vai indo para a direita.

div.gallery:hover { border: 1px solid #777; }

Borda quando passar o mouse por cima.

div.gallery img { width: 100%; height: auto; }

Espaçamento da imagem cobrindo 100% de sua borda e ajuste automatico de altura.

div.desc { padding: 15px; text-align: center; }

Espaçamento do texto para a imagem e alinhamento do texto.
