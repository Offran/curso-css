Bordas

Bordas emcapsulam o elemento com bordas de diferentes tipos.
border-style: propriedade especifica qual borda exibir.
Exemplo: p {border-style: dotted;}

  • dotted
    • Borda pontilhada;
  • dashed
    • Borda sólida.
  • double
    • Borda dupla.
  • groove
    • Borda ranhurada 3D. O efeito dependerá do valor da cor da borda.
  • ridge
    • Define uma borda estriada 3D. O efeito depende do valor da cor da borda.
  • inset
    • Borda inserida 3D. Efeito depende do valor da cor da borda.
  • outset
    • Borda inicial 3D. Efeito depende do valor da cor da borda.
  • none
    • Não define fronteira.
  • hidden
    • Define uma borda oculta.

Bordas com width

Para deixar as bordas mais grossas.
Exemplo:

p.um {
border-style: solid; border-width: 10px;
}

p.dois {
border-style: solid; border-width: medium;
}

p.tres {
border-style: dotted; border-width: 2px;
}

p.quatro {
border-style: dotted; border-width: thick;
}

Larguras laterais específicas

Usando border-width o usuário pode classificar até quatro valores, um para cada lado (superior, direita, inferior e esquerda).

Exemplo:
p.cinco {
border-style: dotted;
border-width: 25px 10px 5px 20px;
}
(Isso nos dá: 25px top, 10px na direita, 5 px em baixo e 20 px na esquerda).

Cores nas bordas

Para adicionar cores nas bordas, basta usar o seletor border-color.
Por exemplo:
p.seis {
border-style: solid;
border-width: 5px;
border-color: green
}

Também é possível colorir laterais específicas com: border-color: red green blue yellow; (seguindo o padrão superior, direita, inferior, esquerda).

Nota: O padrão de cores pode seguir o explicado aqui.

Lados individuais

Além de cor e largura, é possível definir diferentes tipos de bordas para cada lado.
Exemplo:

p.sete {
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;
}

O exemplo acima é o mesmo que fazer:
p {
border-style: dotted solid;
}
Dessa forma cima e baixo ficam dotted e direita e esquerda solid.
Basicamente, caso não seja especificado, a borda contrária aquela que foi especificada, ficará igual ela. Por exemplo, se eu fizer 3 estilos diferentes, cima, direita e baixo, o lado da esquerda ficará igual o da direita.

Propriedade de bordas abreviadas

Podemos abreviar as propriedades das bordas:
p {
border-left: 6px solid red;
}
Nesse caso seria uma borda de 6px, sólida e vermelha. Tudo numa só frase. E aqui especifiquei que a borda seria só na esquerda, com border-left.

Bordas arredondadas

As bordas não precisam ficar sempre quadradas, podemos arredondá-las com "border-radius".
A exemplo desta própria borda:
p.oito {border: solid 5px red; border-radius: 10px;}
