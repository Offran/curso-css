Pseudoelementos e combinadores

Pseudoelementos são usados para estilizazr partes específicas de um elemento.
Por exemplo:

  • Estilize a primeira letra ou linha de um elemento.
  • Insira conteúdo antes ou depois do conteúdo de um elemento.

sintaxe dos pseudoelementos:
seletor::pseudoelemento{
propriedade: valor;
}

Pseudoelemento de primeira linha ::first-line
pseudoelemento de primeira letra ::first-letter

Usando esses pseudoelementos podemos variar primeira linha, primeira palavra, adicionar requisitos, igual como foi mostrado aqui.

Como pode ser visto, aqui estou usando o pseudoelemento ::selection, ao qual seleciono apenas uma parte para ser modificada quando selecionada. Faça a seleção do texto inteiro para ver qual parte está modificada.
div#selection::selection {
color: red;
background: yellow;
}

Combinadores Css

Um seletor css pode conter vários tipos de seletores apra especificar/dar alvo melhor em cada elemento.
Existem quatro combinadores

  • Seletor descendente (espaço)
    • div p {color: blue} Para aplicar em todos os p descendente de divs.
  • Seletor filho (>)
    • div > p {color: blue} Ele é aplicado assim como o descendente, mas restrito ao filho.
  • Seletor de irmão adjacentes (+)
    • div + p {color: blue} Os elementos irmãos adjacentes devem ter o mesmo pai e vir imediatamente um após o outro. Isso significa que se tiver como no exemplo citado, um p depois do p, ainda que irmão, não será referenciado.
  • Seletor geral de irmãos (~)
    • div ~ p {color: blue} Seleciona todos os irmãos que vierem após o elemento referenciado.
