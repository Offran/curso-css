Pseudoelementos

Pseudoelementos são usados para estilizazr partes específicas de um elemento.
Por exemplo:

  • Estilize a primeira letra ou linha de um elemento.
  • Insira conteúdo antes ou depois do conteúdo de um elemento.

sintaxe dos pseudoelementos:
seletor::pseudoelemento{
propriedade: valor;
}

Pseudoelemento de primeira linha ::first-line
pseudoelemento de primeira letra ::first-letter

Usando esses pseudoelementos podemos variar primeira linha, primeira palavra, adicionar requisitos, igual como foi mostrado aqui.

Como pode ser visto, aqui estou usando o pseudoelemento ::selection, ao qual seleciono apenas uma parte para ser modificada quando selecionada. Faça a seleção do texto inteiro para ver qual parte está modificada.
div#selection::selection {
color: red;
background: yellow;
}
