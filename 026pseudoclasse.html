Pseudoclasse

Pseudoclasses são usadas para definir um estado especial de um elemento.

  • Estilizar um elemento quando o usuário passa o mouse por ele
  • Estilize links visitados e não visitados de maneira diferente
  • Estilize um elemento quando ele receber foco

Sintaxe das pseudoclasses:
selector:pseudo-class {
property: value;
}

Pseudoclasse :first-child

Ela serve para deixar o primeiro filho de cada elemento estilizada.
Por exemplo:
Se for feito p:first-child {color: blue} todos os primeiros p que forem filhos de algum outro elemento ficarão azuis. O primeiro p do body, caso tenha um p dentro de uma div, o primeiro p dessa div e assim se segue, sempre o primeiro p de cada elemento.
