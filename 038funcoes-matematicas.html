Funções matemáticas Css

A função calc() CSS permite você executar cálculos quando especificar os valores de propriedades no CSS.
A função calc() recebe uma simples expressão como parâmetro e o resultado desta expressão é utilizado como valor.
Com essa função podemos definir que tal estilização ocupe x espaço dentro de uma largura, altura, etc.

No caso dessa página foi utilizado:

* {
margin: 0;
padding: 0;
box-sizing: border-box;
}

Apliquei um reset para o melhor entendimento.

body { height: 100vh; }

Defini a altura do body.

#header { background-color: rgb(196, 196, 245); height: 60px; width: 100%; }

Defini quantos px teria o #header, junto com quanta largura ele pegaria.

#content {
background-color: rgb(218, 172, 172);
height: calc(100vh - 90px);
width: 100%;
padding: 20px;
}

Defini que o #content cobriria praticamente toda minha altura -90px.

#footer {
background-color: rgb(180, 180, 180);
color: white;
height: 30px;
}

Defini quantos px teria meu #footer. Como pode ser observado, isso foi definido de maneira mais fixa, portanto congelando as áreas definidas em seus tamanhos.
Nota: pode ser usado soma, divisão, multiplicação e subtração normalmente em calc().
Tem que ter muito cuidado, pois isso desformata facilmente o site.



  • calc().
    • Com calc() a estilização é feita diretamente de escolha de quem estiver fazendo o design da página.
  • max().
    • Com max() o usuário irá pegar o maior valor de uma lista definida para dar alvo. Por exemplo: Se uma lista for definida como max(50%, 300px), a que for de maior valor será a selecionada para a estilização.
  • min().
    • Com min() o usuário irá pegar o menor valor de uma lista definida para dar alvo. Por exemplo: Se uma lista for definida como min(50%, 300px), a que for de menor valor será a selecionada para a estilização.
