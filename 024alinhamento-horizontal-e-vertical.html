Alinhamento vertical e horizontal

margin: auto; Para centralizar elementos horizontalmente em um bloco (como div).
Usando text-align: center; Para centralizar o texto (conteúdo) do bloco.
Usando width é possível ver que o elemento foi centralizado a partir de seus limites para o centro.
div#exemplo1{
width: 50%;
overflow: auto;
border: solid 1px green;
margin: auto;
text-align: center;
img { display: block; margin-left: auto; margin-right: auto; width: 40%; }
}

Com line-height podemos centralizar verticalmente.

Cada linha aqui tem o espaçamento previsto de 30px;
div#exemplo2{width: 50%;
overflow: auto;
margin: auto; text-align: center;
border: solid 1px red;
line-height: 30px;
}

Centralizar verticalmente com flexbox

Flexbox é o modelo adotado, usado junto com justify-content para centralizar.
width: 50%;
margin: auto;
overflow: auto;
border: solid 1px blue;
align-items: center;
display: flex;

justify-content: center;
