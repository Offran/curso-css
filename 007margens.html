Margens

Margens criam espaço ao redor dos elementos.

Pode ser feito:
margin-top
margin-right
margin-bottom
margin-left
Como na imagem sugere.

Todas as propriedades de margem podem ter os seguintes valores:

  • auto
    • Navegador calcula a margem.
  • length
    • especificar px, pt, cm, etc,. Inclusive valores negativos são permitidos, porém isso pode fazer as margens invadirem outros espaços de outros elementos.
  • porcentagem (%)
    • Especifica uma margem em porcentagem da largura do elemento que a contém.
  • inherit
    • A margem herda a especificação do elemento pai.

Margem pode ter seus valores e propriedades abreviados:
obs: todas as regras para abreviações de valores e propriedades são válidas.

p {
margin: 25px 50px 75px 100px;
background-color: aqua;
border: solid 2px red;
}

O valor automático da margem

O valor automatico colocará o elemento inteiro centralizado.
p#exemplo2 {margin: auto;
background-color: rgb(92, 111, 111);
border: solid 2px rgb(77, 18, 18);
width: 600px;
}

É importante prestar atenção, que não é o texto, mas sim o elemento inteiro.
Sem o width não faria diferença, pois o elemento estava cobrindo todo seu espaço automaticamente, assim já estando centralizado.

O valor herdado e uma aplicação de exemplo:

O valor herdado pode ser usado quando se quer manter uma constância e não quer perder tempo ou linhas repetindo coisas. Então podemos fazer como no exemplo abaixo:

div {
border: 1px solid red;
margin: 50px;
background-color: background-color: rgb(178, 133, 220);;
}
p.ex1 {
margin: inherit;
}

Nesse exemplo, caso seja retirado o p, então o texto não terá margem a partir da borda do div.

Colapso de margem

As margens inferior e superior podem ser vistas como uma única margem. Então caso seja feito margin-bottom: 50px e margin-top: 20px, pode acontecer de ficar apenas os 50px, pois isso já está cobrindo os 20px da outra.
