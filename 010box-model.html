Box model Css

Modelo de caixa css é como uma caixa literal com várias bordas que enquadram um elemento e seu conteúdo. Esse modelo enquadra ate mesmo o elemento inteiro, dando uma margem para ele.

Legenda:
Conteúdo: Conteúdo do elemento.
Preenchimento: Área entre o elemento e a borda, mais aprofundado em padding.
Borda: Área que enquadra/circula o conteúdo, como uma membrana que fecha um elemento, mais aprofundado em borders.
Margem: Área que enquadra/circula o conteúdo após e antes do elemento, mais aprofundado em margins.

Aqui está um exemplo de box model:

p#exemplo1 {
width: 200px; (largura)
border: 15px solid green; (A borda que separa o conteúdo e a margem).
padding: 50px; (espaço entre a borda e o conteúdo).
margin: 20px blue; (espaço antes e depois do elemento, que está antes e apos a borda).
}
