Planos de fundo CSS

Planos de fundo são usados para adicionar efeitos de fundo aos elementos.

  • background-color
    • Dá cor de fundo a um elemento.
    Podemos adicionar opacidade com "opacity", onde "1" é sem opacidade e 0,0 é sem cor.
  • background-image
    • Adiciona uma imagem ao fundo.
    Exemplo:
    p { background-image: url("paper.gif"); }
    Também é possível adicionar várias imagens ao fundo, como no exemplo:
    body {
    background-image: url("img_tree.gif"), url("paper.gif"); background-color: #cccccc;
    }
  • background-repeat
    • Algumas imagens acabam por ficarem repetindo na horizontal, vertical ou em ambos. Para ajustar isso e a imagem de fundo não ficar se repetindo é necessário usar o repeat.
    Exemplo:
    body {
    background-image: url("gradient_bg.png");
    background-repeat: repeat-x;
    }
    Nesse caso está ajustado o x, com "repeat-y" ajusta o y. Caso a imagem duplique ambos os casos, ambos os repeats deverão ser usados.
    Com "no-repeat" a imagem fica uma só, no canto da tela. Por exemplo, ao invés de várias árvores se repetindo, terei só uma vez essas árvores, sem as repetições.
  • background-position
    • Com "background-position" é possível ajustar onde será a posição citada no exemplo acima.
    Exemplo:
    body { background-image: url("img_tree.png");
    background-repeat: no-repeat;
    background-position: right top;
    }
    Isso deixará a imagem no canto superior direito.
  • background-attachment
    • Ela irá especificar se a imagem deverá ser fixa ou acompanhar a rolagem.
    Com "background-attachment: scroll" a imagem não acompanha a rolagem.
    Com "background-attachment: fixed" a imagem acompanha a rolagem.
  • background (propriedade abreviada)
    • É uma forma abreviada de escrever as questões do background, usando apenas "background:" como propriedade.
    Podemos transformar isso:

    body {
    background-color: #ffffff;
    background-image: url("img_tree.png");
    background-repeat: no-repeat;
    background-position: right top;
    }

    Nisso:

    body {
    background: #ffffff url("img_tree.png") no-repeat right top;
    }
