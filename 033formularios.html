Formulários

Formulários, criação de conta, país, idade, preenchimento de dados, etc. Aqui será demonstrado boas e modernas formas de criar formulários.

O formulário acima foi criado usando as seguintes especificações:
input[type=text], select {
width: 100%;
padding: 12px 20px;
margin: 8px 0;
display: inline-block;
border: 1px solid #ccc;
border-radius: 4px;
box-sizing: border-box;
}

selecionando as seleções (select) e os inputs[tipo=texto], ajustei sua largura, espaçamento e margem. Coloquei para o bloco inteiro ser clicável e uma borda estilizada para ajustar sua aparência como uma box.

input[type=submit] {
width: 100%;
background-color: #4CAF50;
color: white;
padding: 14px 20px;
margin: 8px 0;
border: none;
border-radius: 4px;
cursor: pointer;
}

selecionado o input do tipo submit, ajustei a largura, cor verde de fundo, cor da letra, espaçamento, margem, removi a borda, arredondei as bordas e ajustei o cursor para pointer.

input[type=submit]:hover {
background-color: #45a049;
}

selecionando o input[tipo=submit] ajustei para a cor ficar levemente mais escura quando passar o mouse por cima.

div {
border-radius: 5px;
background-color: #f2f2f2;
padding: 20px;
}

selecionando a div inteira, que cobre todos os inputs, ajustei cor de fundo, raio da borda e espaçamento, para dar uma ideis de que essas informações são um conjunto.

Falando sobre alguns tipos de input:

  • input[type=text]
    • selecionará apenas campos de texto
  • input[type=password]
    • selecionará apenas campos de senha
  • input[type=number]
    • selecionará apenas campos numéricos
  • input[type:button]
    • criará um input do tipo botão

Também posso usar value="nome" para colocar um nome previsto.

Entradas focadas:

Posso dar foco ao clicar na input, dando certo destaque a ela, dessa forma: input#exemplodefoco:focus{background-color: red;}
Posso também adicionar background-image com background-repeat: no-repeat; Assim colocando uma imagem de fundo que não fique se repetindo.

Propriedade transition:
É a propriedade usada para animar a largura de entrada quando recebe foco.
input#transexemplo {
width: 130px; transition: width 0.4s ease-in-out;
}

input#transexemplo:focus { width: 100%; }
