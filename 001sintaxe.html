Introdução CSS

Css é uma linguagem de estilização, a qual descreve como os elementos HTML devem ser mostrados na tela.
O Css economiza muito trabalho, pois ele pode estilizar grupos de várias páginas de uma só vez. Além de ser um bom método para deixar o html mais clean, podendo ser feito em um outro arquivo, diferente do html.
Css definirá estilos para as páginas, incluindo design, layout (modos de distribuição e arranjos) e variações de exibição em diferentes dispositivos, também seus tamanhos de tela.
Diferente de html, os arquivos css possuem extensão ".css", vide "style.css".

Sintaxe CSS

A regra de sintaxes consiste em:
Seletor → propriedade → valor
Porém deve ser escrito dessa forma:
Seletor {propriedade: valor; propriedade: valor;}

(É interessante separar, após cada valor de propriedade colocado, uma linha a mais, afins de deixar o código mais claro).
O seletor apontará para o elemento ao qual o usuário deseja estilizar.

Exemplo:
<p> { color: red; text-align: center; }</p>

Nesse exemplo acima eu deixo todas as letras de todos os "p" da cor vermelha e o texto no centro. Contudo, para não ficar tudo em vermelho, eu fiz isso usando o id "exemplo1". Para usar o id é necessário antes colocar um #.
