Preenchimento Css

O padding (preenchimento) serve para preencher um campo em um elemento, criando um espaço entre as bordas do elemento e o conteúdo. Dessa forma:

Esse conteúdo está 50px sendo apertado de cima e de baixo e sendo apertado 500px para cada lado.

p.exemplo1{
padding: 50px 500px;
border: solid black 2px;
background-color: antiquewhite;
}

Box-sizing: border-box;

É comum que width, padding e margin causem confusão entre si. Para definir uma largura (width) verdadeira, o ideal é usar o box-sizing.
O box-sizing: border-box manterá como sendo a largura máxima a definida por ele, então é possível evitar erros usando esse comando.
