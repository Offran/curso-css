Flutuante e claro/Float and Clear

Float
A propriedade flutuante, faz com que, por exemplo, uma imagem flutue do lado de um texto de conteiner.
Float pode ter 4 valores:

  • left
    • para deixar à esquerda do conteiner
  • right
    • para deixar à direita
  • none
    • O elemento não "flutua" será exibido exatamente onde estiver, sendo esse o valor padrão.
  • inherit
    • O elemento herda o valor float de seu pai.

Clear

clear especifica o que deve acontecer com o elemento próximo a um elemento flutuante.

  • left
    • para deixar abaixo dos elementos flutuantes à esquerda
  • right
    • para deixar abaixo dos elementos flutuantes à direita
  • both
    • para deixar abaixo dos elementos flutuantes à direita e à esquerda
  • none
    • esse é o valor padrão, podendo ir para qualquer direção.
  • inherit
    • O elemento herda o valor clear de seu pai.

Usando overflow: auto:
Como pode ver aqui está uma borda que pega a imagem. Essa imagem não seria capturada pela borda sem o código.
#bordaexemplo{
width: 600px;
border: 1px solid;
overflow: auto;
}

É importante que a imagem esteja logo no início do elemento.
