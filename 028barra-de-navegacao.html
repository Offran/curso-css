Barra de navegação

Barra de navegação é para um melhor design de site, mais moderno e compacto.

Uma barra de navegação é feita usando uma lista de links. Aqui estará uma forma de criação de uma.
ul#remocao {
list-style-type: none;
margin: 0;
padding: 0;
}

Aqui criei uma lista sem margem, sem espaço e sem estilização. Então é uma lista só na semântica, mas não mais na aparência, por enquanto e o resultado está abaixo.

Ajustando a barra de navegação para a vertical

Para fazer isso é preciso estilizar os elementos "a", que estarão com os links das barras de navegação.

#listaembloco{
display: block;
width: 90px;
background-color: gray;
}

Então, adicionando o elemento "a" e essa estilização, ajustando a sua largura para um espaço mais econômico, sua cor para ter destaque e colocando como um bloco, agora todo elemento é clicável, ficando esse o resultado:

Modernizando a lista

Após isso, para uma modernização da barra de navegação, o ideal é ajustar cor ao passar o mouse por cima e remover a aparência de link (sublinhado), dessa forma:
ul#estilizacaoul {
list-style-type: none;
margin: 0;
padding: 0;
width: 200px;
background-color: #f1f1f1;
}

Respectivamente: removi a aparência de lista, zerei a margem, espaçamento, ajustei largura e dei cor de fundo.

li a#estilizacao {
display: block;
color: #000;
padding: 8px 16px;
text-decoration: none;
}

Respectivamente: Ajustei como bloco clicável, coloquei cor na letra, ajustei espaçamento entre o conteúdo e as bordas do elemento e removi decoração de link.

li a#estilizacao:hover {
background-color: #555;
color: white;

}

Respectivamente: hover está fazendo ser selecionado apenas quando passar o mouse por cima, então muda a cor de fundo e a cor da letra.

Barra de navegação horizontal

Como mostrado na imagem de demonstração, a barra de navegação horizontal é simplesmente uma barra que se mantém na mesma linha. in-line. Para fazer uma barra de navegação horizontal basta aplicar

li#inline { display: inline; }
