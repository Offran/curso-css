Ícones Css

Para os ícones, é possível usar acessando o fonte awesome, contudo, não irei aprofundar isso, pois é um estudo a parte e muito grande. Sua lista mais detalhada pode ser vista aqui: tutorial fonte awesome css.

Para usar os ícones bootstrap, adicionar no head o link:
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
Usando <i class="especificação"> pode digitar algo aqui, ou não, para ficar do lado da imagem.</i>
