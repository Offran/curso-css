Índice Css

Quando alguns elementos são posicionados eles podem se sobrepor a outros elementos. Como por exemplo, nessa imagem, seu índice z é -1, então ela está atrás do texto.

img {
z-index: -1;
position: absolute;
left: 350px;
top: 0px;
}

Ainda usando o índice z-index, podemos fazer um elemento de ordem maior sobrepor outro, com índices respectivos de 2 e 1.
Caso dois elementos não possuam índice Z, então o último elemento colocado estará acima.
