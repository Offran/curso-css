Listas Css

Listas, assim como em html, porém aqui elas serão estilizadas.
Em html temos listas não ordenadas (ul) e listas ordenadas (ol).
As propriedades das listas em CSS permitem:

  • Definir diferentes marcadores de itens de lista para listas ordenadas (ao invés de "1. 2. 3.", ser algo como "I II III")
  • Definir diferentes marcadores de itens de lista para listas não ordenadas (ao invés de círculos serem quadrados, etc)
  • Definir imagem como marcador de item da lista
  • Adicionar cores de fundo a listas e itens da lista

Diferentes marcadores de tipos de lista

  • ul.a{list-style-type: circle;}
  • ul.b{list-style-type: square;}
  1. ol.c{list-style-type: upper-roman;}
  1. ol.d{list-style-type: lower-alpha;}
  • ul.e{list-style-image: url(https://cdn.icon-icons.com/icons2/2367/PNG/512/check_small_icon_143632.png);}

list-style-position: out/inside

Ela posiciona os marcadores dentro ou fora das box, como no exemplo abaixo. Por exemplo, se eu coloco dentro de uma borda, as que são "fora da box", os marcadores estarão fora das bordas, enquanto os dentro da box, os marcadores estarão dentro.

  • fora da box 1
  • fora da box 2
  • fora da box 3
  • dentro da box 1
  • dentro da box 2
  • dentro da box 3
