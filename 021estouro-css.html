Estouro/overflow

overflow é a propriedade css que controla o que acontece com um conteúdo grande demais para caber em determinada área. Exemplo abaixo:

p#overflowexemplo {
background-color: antiquewhite;
color: rgb(33, 0, 51);
padding: 10px;
width: 250px;
height: 100px;
overflow: scroll;
border: 1px solid #dd7474;
}

A overflow propriedade possui 4 valores:

  • visible
    • Padrão. O conteúdo é renderizado fora da caixa do elemento.
  • hidden
    • O overflow é cortado e o resto do conteúdo fica invisível.
  • scroll
    • O overflow é ajustado, contendo uma barra de rolagem para verificar todo o conteúdo.
  • auto
    • Semelhante ao scroll, mas adiciona barra de rolagem somente se for necessário.

Invisível: overflow: hidden

A caixa esconte o restante do conteúdo:
p#invisivelexemplo {
width: 200px;
height: 65px;
background-color: rgb(80, 130, 255);
overflow: visible;
}

Visível: overflow: visible

A caixa é pequena demais para conter todo o conteúdo.
O conteúdo inteiro conta com essa estilização:
p#visivelexemplo {
width: 200px;
height: 65px;
background-color: coral;
overflow: visible;
}
