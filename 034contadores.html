Contadores Css

Contadores são variáveis mantidas pelo css cujos valores podem ser incrementados por regras css (para verificação de quantas vezes foram usados). Eles permitem ajustar a aparência do conteúdo com base no seu posicionamento.

Para trabalhar com contadores usaremos os seguintes contadores:

  • counter-reset
    • cria ou redefine um contador
  • counter-increment
    • incrementa um valor de contador
  • content
    • insere conteúdo gerado
  • counter() ou counters()
    • adiciona o valor de um contador a um elemento
Para usar um contador css, primeiro ele deve ser criado com counter-reset.
Aqui abaixo estará um contador do elemento h4, ele é feito da seguinte maneira:
body {
counter-reset: section;
}

aqui eu estou iniciando os números da seção, definindo a sequência pelos h4 que estão no body.

h4::before {
counter-increment: "seção";
content: "Seção" counter(seção) ": ";
}

Respectivamente: (::before) pseudo-elemento filho do elemento atingido.
(counter-increment) está incrementando um valor ao invés de contar do zero.
(content: "Seção ") está criando a palavra que será repetida, inclusive esse espaço extra está dando uma separação para o número do contador.
(counter(seção) ": ";) está selecionando cada seção para o contador não resetar, mas sim adicionar e adicionando os dois pontos no final.

Observação: Pode inclusive contar subcategorias dentro de grandes categorias.
Por exemplo, dentro das h4 eu poderia contar os h5 como subseções (1.1, 1.2, 2.1, 2.2, 2.3... e assim por diante). Só sendo necessário criar um contador de subseção dentro do contador de seção e dando o alvo na subseção h5 para isso.

Aqui estará um exemplo com lista com seções:
ol#secao1 {
counter-reset: section;
list-style-type: none;
}

Em (counter-reset: section;): Aqui eu estou criando o contador de número, definindo por todos os ol#secao1 que citarem section.
Em (list-style-tipo: none;): estou tirando a estilização base da lista.

li#secao1::before {
counter-increment: section;
content: counters(section,".") " ";
}

Em (counter-increment: section;): estou incrementando um valor ao invés de contar do zero.
Em (content: counters(section, ".") " ";): estou dando alvo da "section" citado anteriormente e colocando um ponto entre cada separação de subseção. No final de tudo entra um espaço, antes do conteúdo do elemento (que está escrito "item").

  2. item

    2. item
    3. item
